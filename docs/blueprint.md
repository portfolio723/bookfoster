# **App Name**: BookExchange

## Core Features:

- User Authentication: Secure user authentication via Phone OTP (primary) and Email OTP (optional).
- Browse and Filter Books: Allow users to browse books by category, condition, and price range.
- Book Details: Display detailed information about each book, including images, condition, rent/buy prices, and donor information.
- Donor Intake: Enable donors to submit book details and schedule pickups.
- Realtime Chat: Enable realtime chat between users and donors.
- Payment Processing: Integrate Razorpay for secure payment processing and security deposit management.

## Style Guidelines:

- Primary color: Teal (#26A69A) to evoke a sense of trust and knowledge. The teal will be used with black text on a white background.
- Background color: White (#FFFFFF) to maintain a clean, minimal look.
- Accent color: Dark Slate Gray (#2F4F4F) to be used to call out important CTAs.
- Headline font: 'Inter', a grotesque-style sans-serif, for headings.
- Body font: 'Open Sans', a humanist sans-serif, for body copy and UI elements. Note: currently only Google Fonts are supported.
- Use line icons from Heroicons or Tabler for a low visual noise and a modern aesthetic.
- Employ an 8pt grid for consistent spacing and paddings throughout the app (16/24/32).
